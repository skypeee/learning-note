# STIL

## 初试

1. 如果没有git请先安装git

```shell
sudo apt-get install git
```


2. 首先需要把ArduPilot及子项目代码拉下来
   
```shell 
git clone https://github.com/ArduPilot/ardupilot.git

git submodule update --init --recursive
```

3. 添加环境变量

```shell
export PATH=$PATH:<ardupilot-code-path>/Tools/autotest/
```
添加至~/.bashrc 并执行
```shell
source ~/.bashrc
```
4. 执行模拟
5. 
-v ArduCopter	仿真多旋翼飞行器
-L KSFO	起飞位置设为旧金山机场

```shell
sim_vehicle.py -v ArduCopter -L KSFO
```

## sim_vehicle 说明

-v VEHICLE 设备类型
```shell
  -v VEHICLE, --vehicle=VEHICLE
                        vehicle type (ArduCopter|Helicopter|Blimp|ArduPlane|Ro
                        ver|ArduSub|AntennaTracker|sitl_periph_universal)
```
- ArduCopter 多旋翼无人机
- ArduPlane 固定翼飞机


-L 位置，如 -L KSFO
```shell
    -L LOCATION, --location=LOCATION
                        use start location from Tools/autotest/locations.txt
```

-m  启动时加载地图模块
```shell
    --map               load map module on startup
```

--enable-fgview 启动FilightGear输出（一个3D展示）
```shell
    --enable-fgview     Enable FlightGear output
```
