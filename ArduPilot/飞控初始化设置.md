# 飞控初始化设置
## 1. 设置机架类型
Mission Planner > Setup Tab > Mandatory Hardware > Frame Type
Mission Planner > 初始化设置菜单 > 必选硬件 > 机架类型

根据电机布局选择机架类型
Frame Class 选择 Quad  X 或者 V型

pluss是电机直接在无人机的前面；X是最常见的，每边有两个电机；V类型是用于无人机前臂较短、后臂较长的类型


[设置机架类型](./images/setup_frame_type.png)

## 2. 初始化调参

Mission Planner > Setup Tab > Mandatory Hardware > Initial Tune Parameters
Mission Planner > 初始化设置菜单 > 初始化调参

设置螺旋桨尺寸（英寸）、电池电芯数量、完全充电的电池电压、完全放电电池电压
电池部分可以自行设置，也可以选择电池化学成分进行自动填充
[电池化学成分](./images/Battery_chemistry.png)

还需要勾选4.0建议设置，包含了电池故障安全设置和电子蔚蓝的高度限制，如果不想限制则忽略
[建议设置](./images/suggest_settings.png)

最后点击计算按钮进行初始化参数计算，并写入固件
[计算参数](./images/calculate_initial.png)

## 3. 设置板子的方向

·Mission Planner > Config Tab > Full Parameter List > search "AHRS_ORIENTATiON"

·Mission Planner > 配置菜单 > 全部参数列表 > 搜索 "AHRS_ORIENTATiON"

只有在特定方向（与制造商朝向不同时）上放板子的时候需要修改

官方说明：相对于该板卡类型标准朝向的整体板卡方向。此设置会旋转惯性测量单元（IMU）和指南针的读数，以便将板卡以任意 90 度或 45 度的角度安装在您的飞行器上。每个选项的标签按照该朝向所对应的旋转顺序命名。此设置将在下次启动时生效。更改后，您需要重新校准飞行器的水平姿态。固件版本 4.2 及更早版本可以使用 CUSTOM（100）旋转来设置 AHRS_CUSTOM_ROLL/PIT/YAW 角度以定义 AHRS 方向。较新版本提供了两个通用自定义旋转选项：Custom 1 和 Custom 2，可通过 CUST_ROT1_ROLL/PIT/YAW 或 CUST_ROT2_ROLL/PIT/YAW 角度进行设置。

[朝向](./images/ahrs_orientation.png)

## 4. 校准加速计

Mission Planner > Setup Tab > Mandatory Hardware > Accel Calibration

Mission Planner > 初始化设置菜单 > 必选硬件 > 加速度计校准

点击校准加速度计，根据提示进行（一般是 左右上下背 这几个方向）
点击校准加速度计水平，将无人机水平放置至平面，点击按钮

[加速度计校准](./images/accel_calibration.png)

体积大的无人机需要做轴校准，但是一般小型无人机不用做

## 5. 校准指南针
Mission Planner > Setup Tab > Mandatory Hardware > Compass

Mission Planner > 初始化设置菜单 > 必选硬件 > 指南针

点击 start ，建议是用一根较长的usb线，原理磁铁进行

拿起无人机保持水平
1. 进行360度旋转，缓慢向左开始
2. 然后向右再转360度
3. 向左倾斜90度，然后水平转360度，然后向右转回来
4. 放到右侧，然后水平左转360度，然后向右转回来
5. 将无人机头朝下。水平左转360度，然后向右转回来
6. 将无人机头朝上，水平左转360度，然后向右转回来
7. 将无人机反转，水平左转360度，然后向右转回来
